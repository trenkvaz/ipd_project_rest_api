# РЕСТ АПИ сервис 

## Запуск и сборка 
#### Необходимо установить или скачать
  - В системе должны быть установлены **Docker** и **Node.js**

#### Без среды разработки из командной строки
- Установка
  - Скачать папку с проектом с Гитлаб
  - В папке с проектом в терминале команда **npm install** установит все необходимые модули
  - Команда **npm run build** скомпилирует код в javascript
  
- Тестовый запуск
  - В файле .env выставить переменную **CLEAR_DB=true** это будет очищать базу данных при каждом запуске, чтобы было проще делать тестовые запросы
  - Команда **npm run test:orders** запускает тест запросов к апи заказов
  - Команда **npm run test:users** запускает тест запросов к апи пользователей

- Рабочий запуск
  - В файле .env выставить переменную **CLEAR_DB=false**
  - Команда **npm run docker:up** запускает приложение в Докере
  - Команда **npm run docker:down** останавливает приложение в Докере

- Внесение изменений
  - Если были изменения в коде приложения они подтверждаются командой **npm run build**
  - Если были изменения в Докерфайле или докер-композит, то они подтверждаются командой **npm run docker:up build**
 - 
### Описание АПИ
